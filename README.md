### Method 1: One-Click Deploy (Heroku)

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/midknightmantra/Vesperr)

### Method 2: Local Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/midknightmantra/Vesperr.git
    cd Vesperr
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Get a Session ID:**
    *   [Session Generator](https://midknightmantra-pair.onrender.com) (Scan QR or Pair Code)

4.  **Configure:**
    *   Create a `.env` file (copy from `.env.example`).
    *   Add your `SESSION_ID`.

5.  **Start:**
    ```bash
    npm start
    ```
